'CommonDf.tsv' file contains final alignment information for all ortholog families.

'NormDf.tsv' file contains additional analysis for the substituted ortholog families.

'Columns.txt' file contains column description for the above two files.

All three files are in tab-separated-values format.

The remaining sub-directories have files that are passed on to 'code/Figures.py' for generating tables and the resulting table and figures.
